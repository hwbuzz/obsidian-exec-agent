# Study Group Plan

## 목표
4주 동안 Python 테스트 자동화 기초를 함께 학습한다.

## 운영 방식
- 매주 1회 60분 온라인 모임
- 각자 예제 하나를 실행해보고 회고를 남긴다.
- 모임 전 질문을 문서에 적어둔다.

## 주차별 계획
1. pytest 기본 구조와 assert
2. fixture와 테스트 데이터 준비
3. mocking과 외부 의존성 분리
4. 작은 프로젝트에 테스트 적용

## 완료 기준
- 각자 테스트 파일 3개 이상 작성
- 실패하는 테스트를 읽고 원인을 설명할 수 있음
- 팀 문서에 배운 점과 다음 학습 주제 정리
