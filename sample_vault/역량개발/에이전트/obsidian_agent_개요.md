# Obsidian Agent 개요

## 목적
Obsidian vault 안의 문서와 할 일을 읽고, 질문 응답과 실행 계획 작성을 돕는 개인 실행 지원 agent를 만든다.

## 현재 구현된 기능
- vault 경로 설정
- Gemini API 호출
- Q&A Mode
- Planning Mode
- Planning 결과 markdown 저장

## 아직 구현되지 않은 기능
- Focus Mode
- Priority Mode
- Bottleneck Detector Mode
- Overdue Task Rescheduler Mode
- 독립적인 Action Mode

## 사용 시나리오
- 문서에서 필요한 정보를 빠르게 찾는다.
- 목표를 실행 가능한 작업 목록으로 바꾼다.
- 계획 결과를 markdown 노트로 남긴다.

## 다음 개발 후보
Focus Mode를 추가해서 사용자가 확보한 시간 안에 할 수 있는 작업을 추천한다.
